# Rick and Morty ETL Pipeline

Простой и эффективный процесс сбора данных из API и сохранения их в базу данных.

## 🛠 Технологии
* **Python 3** (библиотеки `requests`, `psycopg2`)
* **PostgreSQL** (хранение данных)
* **Docker & Docker Compose** (быстрое развертывание)

## 🚀 Как это работает
1. **Extract**: Скрипт обращается к Rick and Morty API и получает данные о персонажах.
2. **Transform**: Данные очищаются и подготавливаются для загрузки.
3. **Load**: Скрипт создает таблицу (если её нет) и записывает данные в PostgreSQL.

## 📦 Запуск проекта
С помощью Docker проект поднимается одной командой:
bash
docker-compose up --build
